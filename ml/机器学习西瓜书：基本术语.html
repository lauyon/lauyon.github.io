                                机器学习基本术语整理

                                # 数据集（data set）

                                一组记录的集合，例如：

                                • （色泽=青绿；根蒂=蜷缩；敲声=浊响）
                                • （色泽=乌黑；根蒂=稍蜷；敲声=沉闷）
                                • （色泽=浅白；根蒂=硬挺；敲声=清脆）
                                • ...... 用D={x1,x2,...,xm}D=\lbrace x_1,x_2,...,x_m \rbraceD={x1​,x2​,...,xm​}表示包含mmm个示例的数据集，每个示例有ddd个属性描述： xi=(xi1xi2⋮xid)x_i = \begin{pmatrix} x_{i1} \\ x_{i2} \\ \vdots \\ x_{id}\end{pmatrix}xi​=⎝⎜⎜⎜⎛​xi1​xi2​⋮xid​​⎠⎟⎟⎟⎞​

                                注：ddd又称为样本的维数

                                # 示例（instance）或样本（sample）

                                数据集中每条记录是关于一个事件或对象的描述，例如：

                                • （色泽=青绿；根蒂=蜷缩；敲声=浊响）

                                # 属性（attribute）或特征（feature）

                                反映事件或对象在某方面的表现或性质的事项，例如：

                                • 色泽
                                • 根蒂
                                • 敲声

                                # 属性值（attribute value）

                                属性上的取值，例如：

                                • 青绿
                                • 乌黑
                                • ......

                                # 属性空间（attribute space）或样本空间（sample space）或输入空间（input space）

                                属性张成的空间，例如： 假设有三种属性：色泽、根蒂、敲声，就可以张成一个用于描述细化的三维空间，每个西瓜都可以在这个空间中找到自己的坐标位置。 用XXX表示，xi∈Xx_i\in Xxi​∈X

                                # 特征向量（feature vector）

                                上面说了每个西瓜可以在属性张成的空间中找到自己的位置，也就意味着每个西瓜对应着一个坐标向量，而每个西瓜又是一个示例，一个示例就是一个特征向量。

                                # 学习（learning）或训练（training）

                                从数据中学得模型的过程，这个过程可以通过执行某个算法来完成

                                # 训练数据（training data）

                                训练过程中使用的数据

                                # 训练样本（training sample）

                                训练数据中的每个样本称为一个训练样本

                                # 训练集（training set）

                                训练样本的集合

                                # 假设（hypothesis）

                                学习模型学得的关于数据的潜在规律

                                # 标记（label）

                                带有结果信息的训练样本，例如： （（色泽=青绿；根蒂=蜷缩；敲声=浊响），好瓜）

                                # 样例（example）

                                拥有标记信息的示例，例如： （（色泽=青绿；根蒂=蜷缩；敲声=浊响），好瓜） 一般用(xi,yi)(x_i,y_i)(xi​,yi​)表示第iii个样例

                                # 标记空间（label space）或输出空间（output space）

                                所有标记的集合，用YYY表示

                                # 泛化（generalization）

                                学得模型适用于新样本的能力
